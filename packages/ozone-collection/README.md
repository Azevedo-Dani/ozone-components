 # \<ozone-collection\>
 

 
 
 ## install
 
 ```
 $ npm install --save ozone-collection
 ```
 
 ## usage
 
 ```html
         <ozone-collection
         id="mosaicCollection"
         items="{{searchResults}}"
         total="{{total}}"
         collection="video"
         data-remain="{{dataRemain}}"></ozone-collection>
 ```
